# neovim
- <span style="color: orange; ">First, install the Node.js and Python3 Neovim packages globally. (run 'npm install -g neovim' and 'python -m pip install neovim' in the shell)<br> Then, set the vim.g.node_host_prog to point to the 'cli.js'. (vim.g.node_host_prog = "..../bin/neovim-node-host"(path to cli.js in the Node.js Neovim module))</span>

### REQUIREMENTS
- neovim >= 0.12.0
- lua >= 5.1.5 (needs to be built with LuaJIT)
- luarocks >= 3.13.0
- Git >= 2.53.0
- fzf >= 0.70.0
- fd >= 10.4.2
- ripgrep >= 15.1.0
- tar >= 3.8.4
- curl >= 8.18.0
- node >= 24.13.0
- python = 3.13
- tree-sitter-cli >= 0.26.1 (run 'npm install -g tree-sitter-cli' and 'tree-sitter init-config' on your shell)
- cmake >= 4.3.0
- GNU make >= 4.4.1
- a C compiler
- a Nerd font
- gitui >= 0.28.1
- glow >= 2.1.2

<span style="color: aqua; ">For Unix Systems<br>
- unzip(1)
- gzip(1)
</span>

<span style="color: lightgreen; ">For Windows Systems<br>
- pwsh
- 7zip
</span>
